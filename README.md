# Python for Cyber Security

This repository contains my coursework and lab implementations for the **Python for Cyber Security** course.

The projects in this repository demonstrate fundamental cybersecurity programming concepts using Python, including:

- Socket programming and port scanning
- File integrity verification using cryptographic hashing
- Responsible web scraping for security intelligence
- Automation of security workflows with logging

Each lab focuses on a different cybersecurity concept and includes its own source code, documentation, and outputs.
