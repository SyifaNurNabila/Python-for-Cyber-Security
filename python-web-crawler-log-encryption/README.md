# Web Crawler and Log Encryption (Python)

This project was developed as part of the **Python for Cyber Security** assignment.

The objective of this project is to create a simple Python-based program that can crawl a website, collect information from several pages, save the results into a log file, and then encrypt the log file for security purposes.

The target website used in this project is:

https://www.upj.ac.id

The program collects several links from the website, extracts the **title of each page**, and stores the results in a log file. After that, the log file is encrypted using a cryptography library.

## Project Features

This program performs several main processes:

1. **Web Crawling**  
   The program collects several links from the target website.

2. **Web Scraping**  
   It extracts the title of each webpage that was found during the crawling process.

3. **Logging Results**  
   The extracted information is stored in a log file.

4. **File Encryption**  
   The log file is encrypted using **Fernet encryption** from the `cryptography` library.

## How to Run the Program

### 1. Run the crawler and scraper

```
python src/main.py
```

This step will:

- crawl the target website  
- extract page titles  
- save the results into `outputs/results.log`

---

### 2. Encrypt the log file

```
python src/encryptor.py
```

This step will:

- read the `results.log` file  
- encrypt the contents  
- generate a new file called `results.enc`
